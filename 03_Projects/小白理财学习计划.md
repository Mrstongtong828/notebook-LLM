# 小白理财学习计划

## 目标

用 30 天建立基础理财框架：看清现金流、建立应急金、理解风险和复利，并开始学习适合普通人的长期投资方式。

核心原则：

- 先保命，再增值。
- 先学习，再投入大钱。
- 不借钱投资。
- 不碰看不懂的产品。
- 不相信“稳赚”“内幕”“老师带单”。

## 第 2 周：建立安全垫

目标：先准备应急金，避免一有意外就被迫借钱或卖出投资。

- [ ] 单独设置一个应急金账户
- [ ] 先存下第一笔小额应急金
- [ ] 计算 1 个月必要生活费
- [ ] 设置每月自动转入应急金的金额
- [ ] 目标逐步存到 3-6 个月必要生活费

应急金规则：

- 只用于失业、生病、维修、家庭紧急支出。
- 不用于购物、旅游、投资加仓。
- 用掉后优先补回。

## 第 3 周：学习理财核心概念

目标：理解投资前必须知道的基础词汇。

- [ ] 现金流：收入减支出
- [ ] 资产：可能保值、增值或产生收入的东西
- [ ] 负债：未来需要持续还钱或花钱的东西
- [ ] 复利：收益继续产生收益
- [ ] 通胀：钱的购买力下降
- [ ] 风险：收益不确定，可能亏损，也可能长期回不来
- [ ] 分散投资：不要把钱放在单一资产上
- [ ] 流动性：需要用钱时能不能快速变现

本周练习：

- [ ] 用自己的话解释“风险承受能力”

## 第 4 周：认识常见投资工具

目标：先认识工具，不急着买。

学习顺序：

- [ ] 银行存款：适合现金管理和应急金
- [ ] 货币基金：适合短期闲钱，但不是完全无风险
- [ ] 债券/债券基金：通常波动低于股票，但也会亏
- [ ] 指数基金/ETF：适合长期学习和小额定投
- [ ] 股票：波动大，需要更多研究
- [ ] 保险：主要用于风险转移，不是优先当投资

暂时不碰：

- [ ] 杠杆
- [ ] 期货
- [ ] 期权
- [ ] 高收益理财
- [ ] 不透明私募
- [ ] 不懂规则的加密资产

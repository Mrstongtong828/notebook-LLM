你已经装好了：WSL Ubuntu 里有 `tmux 3.4`。

使用方式：

```powershell
wsl
```

进入 Ubuntu 后运行：

```bash
tmux
```

常用操作：

```bash
tmux new -s codex     # 新建会话
tmux ls              # 查看会话
tmux attach -t codex # 重新进入会话
```

在 `tmux` 里默认快捷键前缀是 `Ctrl+b`。例如：

```text
Ctrl+b 然后按 d   # 暂时退出会话，后台继续运行
Ctrl+b 然后按 %   # 左右分屏
Ctrl+b 然后按 "   # 上下分屏
```

你截图里的 `brew install tmux` 是 macOS/Homebrew 用法；在你这台 Windows 机器上，正确入口是 WSL 里的 Ubuntu，而且已经可用了。

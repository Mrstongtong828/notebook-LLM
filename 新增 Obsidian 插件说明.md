

这份文档记录当前保留的核心插件，以及已经删除的重复性插件。精简原则是：优先保留对大学学习、项目研究、写作整理和知识库维护最有用的插件，删除功能重叠、维护价值低或容易造成同步冲突的插件。

## 当前保留的核心插件

### Dataview

把整个知识库当成数据库来查询。

适合用来：
- 汇总课程作业、项目任务、阅读清单和论文资料。
- 按标签、文件夹、日期、状态自动生成列表。
- 给项目研究做结构化看板和资料索引。

### Templater

比官方模板更强的动态模板系统。

适合用来：
- 自动生成课程笔记、读书笔记、项目日志、会议纪要。
- 插入日期、标题、文件名等变量。
- 和 `QuickAdd`、`Journals` 配合，统一每天和每类笔记的格式。

### QuickAdd

快速捕捉和创建笔记。

适合用来：
- 一键记录灵感、任务、课堂问题和项目想法。
- 快速创建指定类型的笔记，并套用模板。
- 减少重复新建文件、选模板、改标题的步骤。

### Omnisearch

增强全文搜索。

适合用来：
- 在大量课程、项目和资料笔记中快速找内容。
- 查找以前写过但忘记位置的概念、引用、任务或人名。

### Obsidian Git

给知识库做版本备份。

适合用来：
- 保存笔记历史版本。
- 避免误删、误改后无法恢复。
- 给长期项目研究保留稳定备份。

### BRAT

安装未上架或测试版插件。

适合用来：
- 安装 GitHub 上的 Obsidian 插件。
- 管理还没进入官方插件市场的插件。

注意：只建议安装来源可信、确实需要的插件。

### Claudian

把 AI 工作流接入 Obsidian。

适合用来：
- 让 AI 帮你整理笔记、分析资料、生成总结。
- 配合项目研究和长期知识库维护使用。

### Terminal

在 Obsidian 内打开终端。

适合用来：
- 运行脚本、Git 命令或本地工具。
- 配合 AI/项目研究处理文件和自动化任务。

### Notebook Navigator

增强文件导航和笔记浏览。

适合用来：
- 更高效地浏览课程、项目、模板和资料文件夹。
- 替代部分文件夹美化类插件的需求。

### Excalidraw

手绘白板和视觉化思考工具。

适合用来：
- 画流程图、系统图、思维导图、课程结构图。
- 把图形和笔记双向链接。
- 做项目方案草图和研究框架图。

### Local Images Plus

把外部图片本地化，并管理图片附件。

适合用来：
- 解决网页剪藏图片失效问题。
- 把远程图片保存到知识库附件目录。
- 配合附件清理工具维护图片资源。

### Commander

自定义按钮、工具栏和命令入口。

适合用来：
- 把常用命令放到顶部、侧边栏或移动端工具栏。
- 给“新建课程笔记”“打开项目看板”“快速捕捉”等动作做成按钮。

### Clear Unused Images

清理没有被笔记引用的图片附件。

适合用来：
- 删除剪藏、截图、导入资料后遗留的无用图片。
- 控制知识库体积。

注意：清理前建议先用 `Obsidian Git` 备份。

### Media Extended

增强音频、视频播放和时间戳笔记。

适合用来：
- 边看网课、讲座、访谈边做笔记。
- 在笔记中记录视频时间点，之后跳回对应片段。

### Douban

从豆瓣导入书籍、电影、音乐等资料信息。

适合用来：
- 建立读书笔记和观影记录。
- 自动补充封面、作者、评分等元数据。
- 管理课程阅读、通识资料和研究参考书单。

### Note Refactor

把长笔记拆成多篇独立笔记。

适合用来：
- 把课堂长笔记拆成知识点卡片。
- 把会议纪要拆成任务、问题和方案。
- 把临时摘录加工成可链接的小主题。

### Journals

管理日记、周记、月记等周期性笔记。

适合用来：
- 写每日学习记录、每周复盘和月度总结。
- 跟踪课程进度、项目推进和个人计划。

### Hover Editor

把悬浮预览变成可编辑的小窗口。

适合用来：
- 一边读主笔记，一边修改引用笔记。
- 文献阅读、资料对照时快速补充内容。

### File Explorer Note Count

在文件管理器中显示每个文件夹的笔记数量。

适合用来：
- 快速看出课程、项目、主题下积累了多少笔记。
- 辅助整理知识库结构。

### Smart Connections

自动发现当前笔记和知识库中其他笔记的语义关联。

适合用来：
- 写论文、报告、项目总结时发现相关旧笔记。
- 帮助建立双链，减少重复记录。
- 做主题研究和期末复习时寻找隐藏关联。

### Kanban

看板式任务和项目管理。

适合用来：
- 管理课程作业、项目任务、论文流程。
- 按“待做 / 进行中 / 已完成”组织任务。

### Full Calendar

日历视图和事件管理。

适合用来：
- 管理课程、DDL、考试、项目节点。
- 替代普通 `Calendar` 插件和部分日程插件。

## 已删除的重复性插件

### Calendar

删除原因：和 `Full Calendar`、`Journals` 的日期管理能力重叠。当前保留 `Full Calendar` 负责日程，`Journals` 负责周期笔记。

### Day Planner

删除原因：和 `Full Calendar`、`Kanban`、`Journals` 在计划管理上重叠。为了减少日程入口混乱，暂时不保留。

### TimeStamper

删除原因：功能较单一，时间戳可以通过 `Templater`、系统快捷输入或模板变量解决。

### File Color

删除原因：主要是文件夹视觉美化，对学习和项目研究的核心收益较低，而且容易和主题、导航插件形成视觉负担。

### Remotely Save

删除原因：和 `Obsidian Git` 在备份/同步上容易产生职责重叠。当前优先保留 `Obsidian Git` 做版本备份，避免多个同步工具同时操作同一个库导致冲突。

## 推荐工作流

### 课程学习

- `QuickAdd` 新建课堂笔记。
- `Templater` 自动套用课程模板。
- `Media Extended` 记录网课时间点。
- `Note Refactor` 课后拆分知识点。
- `Dataview` 汇总作业、考试和复习清单。

### 项目研究

- `Kanban` 管理任务阶段。
- `Dataview` 汇总资料、问题和产出。
- `Excalidraw` 画研究框架和方案图。
- `Smart Connections` 找相关旧笔记。
- `Journals` 写项目日报和周报。

### 知识库维护

- `Notebook Navigator` 浏览和整理文件夹。
- `File Explorer Note Count` 观察笔记分布。
- `Local Images Plus` 本地化图片。
- `Clear Unused Images` 定期清理无用附件。
- `Obsidian Git` 保存版本备份。

### Windows 接入 Hermes

> Hermes 是 Meta 开源的 JavaScript 引擎，专为 React Native 优化，启动快、内存省。支持 Windows 本地编译 & React Native for Windows。

- [ ] 了解 Hermes 定位：JS 引擎 vs V8/JSC 的区别
- [ ] 方式一（推荐）：通过 React Native for Windows 接入——Hermes 已默认启用
- [ ] 方式二：在 Windows 本地编译 Hermes CLI（需 Visual Studio + CMake）
- [ ] 尝试运行 Hermes 字节码编译 (`hermesc`) 或 REPL
- [ ] 记录踩坑过程和心得

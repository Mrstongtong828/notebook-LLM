## 项目清单

### 1. Claude Code / Claude Code 插件生态

- 链接：
  - https://github.com/anthropics/claude-code
  - https://github.com/anthropics/claude-plugins-official
- 类型：AI 编程 Agent / Claude Code 插件生态
- 简介：Claude Code 是 Anthropic 的 AI 编程助手，面向真实代码库中的修 bug、重构、实现功能、运行命令和管理开发流程。`anthropics/claude-code` 更像官方 issue / 反馈入口；`claude-plugins-official` 是 Anthropic 管理的 Claude Code 插件目录。
- 适合我的原因：可以作为理解 Codex、Claude Code、Cursor 等 AI 编程 Agent 工作流的参照系，尤其适合研究“插件、权限、沙箱、任务委派、代码库上下文”这些问题。
- 下一步：
  - [ ] 对比 Codex 自动化和 Claude Code 插件机制。
  - [ ] 单独整理 `02_Wiki/AI 编程 Agent 工作流.md`。

### 2. CodeGraph

- 链接：https://github.com/colbymchenry/codegraph
- 类型：本地代码知识图谱 / Agent 上下文增强
- 简介：CodeGraph 会为代码库建立本地知识图谱和索引，让 Claude Code、Codex、Cursor、Gemini 等 Agent 不必反复 grep/read 全仓库，而是通过预先索引的结构化图谱理解代码关系。官方 README 强调它是 100% local，并用于减少 token、工具调用和探索成本。
- 适合我的原因：非常适合接入你正在用的 Codex 工作流。你的项目多、上下文分散，CodeGraph 这类工具可以帮助 Agent 更快理解代码结构。
- 下一步：
  - [ ] 在一个小项目里试跑 `npx @colbymchenry/codegraph`。
  - [ ] 观察它是否能降低 Codex 读文件成本。

### 3. Understand Anything

- 链接：https://github.com/Lum1104/Understand-Anything
- 类型：代码库 / 文档 / 知识库可视化理解工具
- 简介：Understand Anything 使用多 Agent 流水线扫描项目，提取文件、函数、类、依赖关系，并生成可交互的知识图谱和仪表盘。它的定位是“让图谱真正教会你理解项目”，而不是只做炫酷可视化。
- 适合我的原因：它和你的 Obsidian 知识迭代机制很契合。未来可以考虑把项目代码、文档和 Wiki 变成可浏览的知识图谱，用来学习大型项目结构。
- 下一步：
  - [ ] 试用于 `Horizon/` 或一个校园小程序项目。
  - [ ] 评估它生成的图谱是否适合沉淀进 Obsidian。

### 4. Presenton

- 链接：https://github.com/presenton/presenton
- 类型：开源 AI 演示文稿生成器 / Gamma 替代
- 简介：Presenton 是一个开源 AI Presentation Generator，可以通过 AI 生成演示文稿，并提供 API。它的定位类似 Gamma、Beautiful AI、Decktopus 的开源替代，强调可自部署、可控模型和数据隐私。
- 适合我的原因：你有课程汇报、项目展示、比赛材料和作品集需求。Presenton 可以用于快速生成项目路演稿、课程 PPT 初稿、作品集介绍页。
- 下一步：
  - [ ] 用它为“校园服务小程序”生成一次路演 PPT 初稿。
  - [ ] 观察生成质量，决定是否纳入常用工具链。

### 5. NVIDIA LongLive

- 链接：https://github.com/NVlabs/LongLive
- 项目页：https://nvlabs.github.io/LongLive/
- 类型：实时交互式长视频生成
- 简介：LongLive 是 NVIDIA Labs 的实时交互式长视频生成项目，采用 frame-level autoregressive 思路，支持长时间、多段提示词驱动的视频生成。项目文档提到它面向 H100/A100 等高端 GPU，硬件门槛较高。
- 适合我的原因：它更适合作为前沿方向观察，不适合马上本地复现。对理解视频生成、实时交互、多模态生成和 GPU 推理优化有价值。
- 下一步：
  - [ ] 只收藏论文/项目页，不急着跑。
  - [ ] 关注是否出现轻量化 demo、Colab 或消费级显卡版本。

## 我的判断

这组项目可以分成三类：

1. **AI 编程 Agent 上下文增强**：Claude Code、CodeGraph、Understand Anything。
2. **知识/内容自动生成**：Presenton、MoneyPrinterTurbo。
3. **前沿多模态生成研究**：NVIDIA LongLive。

对我当前最值得优先尝试的是：

1. `CodeGraph`：直接服务 Codex/Claude Code 代码库理解。
2. `Understand Anything`：适合和 Obsidian 知识库、Horizon 项目结合。
3. `Presenton`：适合课程汇报、比赛展示和作品集。

`LongLive` 更像前沿观察项，`MoneyPrinterTurbo` 适合之后研究内容自动化。
